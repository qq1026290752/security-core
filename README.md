## 项目所达到的愿景
> 打造一个可以用于任何spring boot项目的安全框架。
##本模块主要功能点
- 本模块主要对APP和浏览器项目相同部分进行抽取,对接口进行抽取
- 对于 properties 部分进行管理
## 项目所用框架
-     spring boot
-     spring security
-     spring social
-     spring session
-     redis缓存服务器
